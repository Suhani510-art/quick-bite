#  QuickBite

A full-stack MERN web application that helps users discover, browse, and manage restaurant food deals. The platform enables users to explore available offers, search restaurants, and securely manage their accounts.

---

##  Features

- User Authentication
- Secure Login & Registration
- Browse Restaurant Food Deals
- Search & Filter Deals
- Responsive User Interface
- RESTful API Integration
- MongoDB Database
- JWT Authentication

---

##  Tech Stack

### Frontend
- React.js
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

### Authentication
- JWT (JSON Web Token)

##  Installation

### Clone Repository

```bash
git clone https://github.com/Suhani510-art/quick-bite.git
```

### Install Dependencies

Frontend

```bash
cd client
npm install
npm start
```

Backend

```bash
cd server
npm install
npm run dev
```

---

## Environment Variables

Create a `.env` file inside the server folder.

```
PORT=5000

MONGO_URI=your_mongodb_connection

JWT_SECRET=your_secret_key
```
